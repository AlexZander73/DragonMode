# Dragon Mode navigation note

The approved screen references are the visual source of truth, but their bottom navigation is not fully consistent. Some crops omit a tab, repeat a label, or highlight a destination that belongs under a different primary area.

The built app intentionally standardises one persistent five-tab bar everywhere: Lair, Hoard, Quests, Scrying, and Treasury.

The ten reference areas map into those five destinations through visible in-screen portals: Dragon Status and Dragon's Legacy open from Lair; Dragon's Rest opens from Quests; Flight Path opens from Scrying; Tribute Hall and Debt Chamber live inside Treasury.
